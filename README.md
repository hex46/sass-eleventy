# Projet de test pour l'artice 'SASS & Eleventy'

## Dépendances

- Eleventy
- SASS
- PostCSS
- Autoprefix
- cssnano

## Lancer le projet

La commande de lancement du projet est déclarée dans `package.json` :
```js
[...]
  "scripts": {    
    "serve": "ELEVENTY_ENV=dev npx eleventy --serve"
  },
[...]
```
